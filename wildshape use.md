Spend a Wild Shape use on a subclass feature that costs a use (Wild Companion, Land's Aid, Wrath of the Sea,
Moonlight Step, Starry Form, and so on). Shows the rules text and spends a use; cast or roll the actual
effect with `!cast` or `!attack`.

Wild Resurgence (level 5) is handled directly: it moves Wild Shape uses and spell slots for you.

__Usage__
`!wildshape use` lists available features.
`!wildshape use companion`
`!wildshape use resurgence` spends a spell slot to regain a use (only when you have none left).
`!wildshape use resurgence -slot` spends a use to regain a level 1 slot (once per Long Rest).
